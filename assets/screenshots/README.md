# App Store screenshots

The homepage showcase (`index.html` → `#showcase`) loads four phone screenshots
from this folder. Save the App Store marketing screenshots here with these exact
filenames (PNG):

| Filename | Which screenshot |
| --- | --- |
| `agents-in-pocket.jpg` | "All agents in pocket" — the session list (checkout-service, design-system, …) |
| `voice-mode.jpg` | "Voice coding anywhere" — the voice screen |
| `always-in-control.jpg` | "Always in control" — the permission request (Allow / Session / Deny) |
| `any-session.jpg` | "One tap, any session" — the session switcher dropdown |

Tips:
- Any portrait aspect ratio works (they're shown at equal width, height auto).
  The App Store 6.9" size (1290 × 2796) is ideal — crisp on retina.
- Keep each file a reasonable size (< ~600 KB). If they're large, export as PNG
  at 1× the display size or run them through an optimizer.
- To add or reorder, edit the `.shots` block in `index.html`.
