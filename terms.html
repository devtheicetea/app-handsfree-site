What Handsfree is

Handsfree turns your iPhone into a voice and text client for coding agents — such as Claude Code and Codex — that run on a computer you own and control. It connects to a companion open-source "bridge" program you run on that computer; you speak or type, Handsfree relays your input to the agent, and reads its replies back so you can work hands-free.

Handsfree is a remote interface. It does not run code itself and does not include a coding agent or any AI model. The actual work is performed by the agent on your own computer.

The bridge and your computer
