隐私优先设计

你的语音只留在手机上。

没有云端语音模型,无需额外托管服务,语音也不需要任何 API 密钥。桥接程序只传输文本。

  • 语音输入、语音输出—采用 Apple 设备端识别与合成,音频永不经过网络。
  • 桥接程序归你所有—运行在你自己的机器上,紧挨着你的助手,使用纯文本 WebSocket 协议通信。
  • 没有新账单—语音不会增加任何费用。你的助手使用你已有的订阅运行。

问答

语音真的免费吗?

是的。语音转文字和文字转语音都运行在应用内 Apple 内置的设备端引擎上。语音体验没有云端 API、无需托管语音模型、没有订阅,也没有按分钟计费。

运行它需要什么?

一部装有 Handsfree 应用的 iPhone,以及一台运行 Node 20+ 的电脑,且已安装并登录 Claude Code 和/或 Codex CLI。用 npx handsfree-bridge@latest 启动桥接程序,然后扫描二维码。

可以在外面用吗?

在同一 Wi-Fi 下开箱即用。想随时随地连接你的助手,只需在手机和电脑上安装 Tailscale—桥接程序会自动公布你的 Tailscale 地址,无需端口转发。

让助手执行命令安全吗?

你始终是那道关卡。每次工具调用都会依据你选择的权限模式进行检查,请求会推送到手机,供你选择"允许 / 本次会话允许 / 拒绝"。把桥接程序绑定到局域网或 Tailscale,并可选加上共享密钥,确保只有你自己的设备能访问。

可以用电脑和手机连同一个会话吗?

可以—多部手机能同时查看并驱动同一个实时会话,全程同步。桥接程序作为唯一写入方,你的历史记录永不分叉或损坏。
